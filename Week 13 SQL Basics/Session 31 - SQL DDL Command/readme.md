| Command    | Description                         | Example                             |
| ---------- | ----------------------------------- | ----------------------------------- |
| `CREATE`   | Create new tables or databases      | `CREATE TABLE students (...);`      |
| `ALTER`    | Modify structure of existing tables | `ALTER TABLE students ADD age INT;` |
